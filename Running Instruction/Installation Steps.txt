Installation Steps(Configuration)
1. Download and Unzip file on your local system.
2.Copy LeaveSync folder and LeaveSync folder inside root directory (for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/html)

Database Configuration

Open phpmyadmin
Create Database elmsdb
Import database elmsdb.sql (available inside zip package)
Open Your browser put inside browser “http://localhost/LeaveSync-main%20(1)/LeaveSync-main/”
